Tecnologías Principales

JavaScript

Es un lenguaje de programación interpretado, con este lenguaje he aprendio a programar y he realizado mis más recientes proyectos.

React

Es una herramienta de Front-End muy útil, me siento muy comodo utilizando esta libreria de JavaScript y cada vez aprendo más sobre ella.

Redux

Redux es un contenedor predecible del estado de aplicaciones JavaScript. Es una excelente herramienta para manejar el estado de una aplicación. Con Redux se logra centralizar el estado de la aplicación y por lo tanto unificar el lugar para realizar cambios. Esto hace el desarrollo muchísimo más simple.

Node.js

Node.js es un entorno de tiempo de ejecución de JavaScript que se utiliza para crear aplicaciones escalables del lado del servidor y de red a través de servidores privados virtuales.

Express

Express, es un entorno de trabajo para aplicaciones web para el programario Node.js, de código abierto y con licencia MIT. Se utiliza para desarrollar aplicaciones web y APIs. Express permite definir rutas que corresponden a métodos HTTP como son peticiones GET, POST, PUT, DELETE.

HTML

Es el lenguaje que estructura y despliega una página web, es amigable y sencillo de implementar.

CSS

Es el complemento de toda página web, un lenguaje de diseño para crear la representación visual de un documento estructurado. Un lenguaje sin limites para la creatividad e indispensable para cualquier Web Developer.

Git

Es una herramienta que nos permite controlar las versiones de nuestros aplicativos, además está diseñada para trabajar en equipo. En definitiva, una herramienta esencial a la hora de desarrollar cada uno de los proyectos que formo parte o desarrollo individualmente.

SQL

SQL es un lenguaje de dominio específico, diseñado para administrar, y recuperar información de sistemas de gestión de bases de datos relacionales.

PostgreSQL

PostgreSQL es un sistema de código abierto empleado para gestionar bases de datos de tipo relacional, aquellas que administran datos relacionados entre sí; tablas integradas por registros y campos en la que cada registro tiene una única identidad.
